#### Q42. What do you need to change about this code to get it to run?

```js
class clock extends React.Component {
  render() {
    return <h1>Look at the time: {this.props.time}</h1>;
  }
}
```

- [ ] Remove `this`
- [ ] Remove the render method
- [x] Capitalize `clock`

</details>
